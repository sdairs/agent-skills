# Sections

This file defines all sections, their ordering, and descriptions.
The section ID (in parentheses) is the filename prefix used to group guides.

---

## 1. Setup (setup)

**Description:** Install the chv CLI and a ClickHouse version. Required before any local development work.

## 2. Local Development (local)

**Description:** Initialize project-local ClickHouse environments, run a local server, and run the client to execute SQL from files using chv. Essential for development and testing before deploying to cloud.

## 3. Migration (migrate)

**Description:** Migrate schemas and data from a local ClickHouse development environment to a ClickHouse Cloud service.
