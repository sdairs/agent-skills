# Sections

This file defines all sections, their ordering, and descriptions.
The section ID (in parentheses) is the filename prefix used to group guides.

---

## 1. Setup (setup)

**Description:** Configure ClickHouse Cloud API credentials. Required before using any cloud commands.

## 2. Service Management (service)

**Description:** Create, start, stop, delete, and inspect ClickHouse Cloud services. Covers the full service lifecycle.

## 3. Sizing & Scaling (sizing)

**Description:** Configure replica count and memory, idle scaling behavior, and choose the right cloud provider and region for your workload.

## 4. Security (security)

**Description:** Configure IP allowlists and disk encryption options to secure ClickHouse Cloud services.

## 5. Backups (backup)

**Description:** List, inspect, and restore from ClickHouse Cloud service backups.
