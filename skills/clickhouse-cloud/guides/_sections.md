# Sections

This file defines all sections, their ordering, and descriptions.
The section ID (in parentheses) is the filename prefix used to group guides.

---

## 1. Setup (setup)

**Description:** Install the chv CLI and configure ClickHouse Cloud API credentials. Required before using any cloud commands.

## 2. Local Development (local)

**Description:** Initialize project-local ClickHouse environments, run a local server, and execute SQL queries using chv. Essential for development and testing before deploying to cloud.

## 3. Service Management (service)

**Description:** Create, start, stop, delete, and inspect ClickHouse Cloud services. Covers the full service lifecycle.

## 4. Sizing & Scaling (sizing)

**Description:** Configure replica count and memory, idle scaling behavior, and choose the right cloud provider and region for your workload.

## 5. Local to Cloud (migrate)

**Description:** Migrate schemas and data from a local ClickHouse development environment to a ClickHouse Cloud service.

## 6. Security (security)

**Description:** Configure IP allowlists and disk encryption options to secure ClickHouse Cloud services.

## 7. Backups (backup)

**Description:** List, inspect, and restore from ClickHouse Cloud service backups.
